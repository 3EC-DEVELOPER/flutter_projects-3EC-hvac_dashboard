# Contributing

Thanks for your interest in the 3EC HVAC Dashboard project.

This repository is primarily maintained as a public-facing UI and front-end portfolio piece for Three Energy Control (3EC). The codebase is open for discussion and improvement, but contributions should stay aligned with the purpose of the project.

## Before opening a pull request

- Raise an issue first for significant UI, structural, or platform changes
- Keep proposals focused on HVAC/BMS dashboard presentation, responsiveness, maintainability, or documentation quality
- Avoid introducing unrelated framework churn or speculative refactors

## Good contribution areas

- Responsive layout improvements
- Accessibility and usability refinements
- Visual polish that fits the existing BMS/HVAC direction
- Documentation corrections or enhancements
- Minor bug fixes in presentation logic

## Out of scope

- Backend/service integrations not already represented in the project
- Large architectural rewrites without prior discussion
- Branding changes that conflict with Three Energy Control (3EC) identity

## Development notes

- Target modern stable Flutter and Dart releases
- Keep changes readable and production-minded
- Preserve the repo's role as a portfolio-quality UI showcase

## Pull request expectations

- Use a clear title and short summary
- Explain the user-facing impact
- Include updated screenshots when the visual output changes materially
- Keep pull requests focused and easy to review
